# React Components Showcase

Aplicação Node.js + React para apresentar exemplos didáticos de:

- componente funcional com `React.createElement`
- componente funcional com JSX
- componente definido com classe
- props em função
- props em classe
- `defaultProps`
- componente stateless
- componente stateful com função
- componente stateful com classe

## Requisitos

- Node.js 18+ recomendado
- npm 9+ recomendado

## Como executar

```bash
npm install
npm run dev
```

Depois, abra o endereço informado pelo Vite, normalmente `http://localhost:5173`.

## Produção

```bash
npm run build
npm run preview
```
